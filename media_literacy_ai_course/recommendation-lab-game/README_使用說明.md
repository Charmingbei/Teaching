# 推薦怪圈實驗室｜網頁版操作遊戲

## 上課使用方式

1. 開啟 `index.html`，或使用本機網址投影。
2. 點選「開始實驗」。
3. 依序操作四個回合：
   - 第1回合：選一張影片，再選小宇的反應。
   - 第2回合：看AI依照最高分標籤推來的新影片。
   - 第3回合：抽怪圈事件卡，判斷小宇會不會更停不下來。
   - 第4回合：選擇工具卡，幫小宇拿回選擇權。
4. 使用左側組別按鈕切換四組，每組會保留自己的分數、觀看時間與推薦路徑。
5. 各組在「想一想」題目下方輸入小組答案，按「提交本組想法」。
6. 老師到頁面下方「教師後台」，輸入密碼 `888` 查看四組提交內容，進行全班討論。
7. 右側分數表可引導學生觀察：演算法不是讀心，而是根據行為資料推測偏好。

## 教學提醒

- 不要把推薦系統說成一定壞，重點是讓學生理解「它會被我的行為餵養」。
- 第4回合請收束到選擇權：暫停、換關鍵字、設定時間、關閉自動播放。
- 結尾可接第三節「AI分身共創故事」，討論數位分身是否會被我們的觀看紀錄影響。
- 教師後台資料會暫存在同一台電腦的瀏覽器中；需要重新收答案時，輸入密碼 `888` 進入後台，再按「清除提交紀錄」。
